# Proof Weekly Template

Use this format for weekly proof publication and internal review.

## Metadata
- Week start: `YYYY-MM-DD`
- Verification date: `YYYY-MM-DD`
- Canonical proof table: `docs/AUDIT.md#71-proof-table-reproducible-claims`

## Claim Entry (repeat per claim)
- Claim: `<short claim text>`
- Command: ``<exact command>``
- Artifact path: ``<repo-relative artifact path>``
- Verification date: `YYYY-MM-DD`
- Delta vs previous week: `<unchanged|improved|regressed|n/a + note>`

## Suggested Baseline Claim Set
- Audit contract is machine-readable.
- Contract snapshot is deterministic.
- Case-study output is machine-readable.
- Proof freshness is enforceable.

## Cadence
- Weekly: publish one proof post.
- Monthly: publish one comparative or case-study post.
